#  Accessibility Widget (A11Y)

A lightweight, plug-and-play accessibility widget that can be added to **any website**
using a single `<script>` tag.

No backend access required.  
No domain required.  
WCAG-friendly foundation.

---

##  Features

-  Font size increase / decrease / reset
-  Multiple contrast modes (high contrast, blue, yellow)
-  Text-to-speech (screen reader)
-  User preferences saved in browser
-  Works on any website (HTML, WordPress, Laravel, CMS)
-  Keyboard & accessibility friendly

---

##  How to Use (One Line Only)

Add this line before `</body>` of your website:

```html
<script src="https://cdn.jsdelivr.net/gh/aditi211202/Web_accessibility/accessibility.js"></script>
